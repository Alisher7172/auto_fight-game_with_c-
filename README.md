# auto_fight-game_with_c++

summary: A small auto fight game written in C++

## Overview

intended for home assignment game starts automatically but you can adjust

## Features

- Simple turn-based auto-fight mechanics.
- Minimal C++ codebase for learning and experimentation.
- Easy to build and run locally.

## Quick Start

1. Clone the repo:
   git clone https://github.com/Alisher7172/auto_fight-game_with_c++.git
2. Change into the project directory:
   cd auto_fight-game_with_c++
3. Run:
   cd "path/to/homework_fight"
   g++ -c *.cpp
   g++ *.o -o Project1.exe
   ./Project1.exe
## Configuration

 no configuration variables
